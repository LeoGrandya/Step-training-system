# 按“第5格启动 → 到n格停止 → 从n格启动还原 → 回到第5格”规则的九宫格步法分析项目

## 1. 本次修改做了什么

本版已经按你的要求改成下面这套逻辑：

1. **新增 `user_params.py`**
   - 所有阈值都在这里手动输入
   - 身高、体重也在这里输入
   - 输入 CSV 路径、输出路径也在这里改

2. **不再沿用旧的通用单位步伐切分**
   - 现在按你给的规则切分：
     - 从 **第5格第一次临界点3（启动后）** 开始分析
     - 临界点1：进入目标格
     - 临界点2：在目标格停止
     - 临界点3：从目标格再次启动
     - 临界点4：回到第5格

3. **输出目录按你要求拆分**
   - `速度参数`
   - `腾空参数`
   - `关节参数`
   - `总体参数`

4. **表格形式的内容输出为 Excel**
   - 逐帧表
   - 状态事件表
   - 状态汇总表
   - 总体参数表
   - 评价参数表

5. **其余按更适合的形式输出为图片**
   - 速度曲线
   - 离地高度曲线
   - 关节角度曲线
   - 力矩曲线
   - 功率曲线

都在：
```python
"thresholds": {
    ...

```

比如：

- `entry_com_frames`：重心进入某格连续多少帧才算进入

- `single_airborne_height_m`：单脚腾空判定高度

## 6. 说明

### 6.1 关于“从第5格第一次临界点3开始分析”
本版已经按这个要求处理：
- 先在第5格找到静止窗口
- 再找第5格第一次启动
- 从这个启动点开始进入正式分析

### 6.2 关于关节力矩
当前采用的是你要求的**简化逆动力学公式**：

```text
关节力矩 = 环节质量 × 质心加速度 × 质心到关节距离 + 环节转动惯量 × 关节角加速度
```

并使用 `身高 + 体重` 估算大腿/小腿质量、长度、转动惯量。

### 6.3 关于“表格形式的以表格形式输出”
已经尽量把：
- 逐帧结果
- 事件结果
- 汇总结果
都输出成 Excel 表。
